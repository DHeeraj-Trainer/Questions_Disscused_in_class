01

Two ways to loop

A loop repeats a block. for repeats a known number of times; while repeats until a condition turns false. Edit either one and run it.

for i in range(5): # i = 0,1,2,3,4 print("row", i)
i = 0 while i < 5: print("row", i) i += 1 # must change i, or it loops forever
02

Number patterns

Load a preset from the dropdown or write your own. The inner loop controls how many numbers print on each row.

for i in range(1, 6): for j in range(1, i + 1): print(j, end=" ") print()
03

Alphabet patterns

Same skeleton — only the printed value changes. Turn a number into a letter with chr(65 + j), because chr(65) is 'A'.

number → letter

chr(65) → 'A'      # chr(66)→'B', chr(67)→'C' …
chr(65 + j) # j = 0,1,2 → A,B,C
for i in range(1, 6): for j in range(i): print(chr(65 + j), end=" ") print()
04

Shape patterns (stars)

Shapes are just spaces and stars. Two questions per row i: how many spaces, how many stars? Try changing "★" to any character.

n = 5 for i in range(n): print(" " * (n - i - 1) + "★" * (2 * i + 1))
05

Hollow patterns

Hollow = print ★ only on the border, blank inside. The border test: first/last row or first/last column.

the border condition

if i==0 or i==n-1 or j==0 or j==n-1:
    print("★", end="")
else:
    print(" ", end="")
n = 5 for i in range(n): for j in range(n): if i == 0 or i == n - 1 or j == 0 or j == n - 1: print("★", end="") else: print(" ", end="") print()
