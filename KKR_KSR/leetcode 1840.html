Peak Formula

peak = (h1 + h2 + distance) // 2

Instead of generating all buildings, we only keep restricted points.

Why Not Build All Buildings?

Approach Memory Time
Generate All n Buildings O(n) O(n)
Process Restrictions Only O(k) O(k log k)

n can be 1,000,000,000. Creating an array of size 1e9 is impossible.

Python Solution

class Solution:
    def maxBuilding(self, n, restrictions):

        restrictions.append([1,0])
        restrictions.sort()

        if restrictions[-1][0] != n:
            restrictions.append([
                n,
                restrictions[-1][1] +
                (n - restrictions[-1][0])
            ])

        m = len(restrictions)

        for i in range(1,m):
            d = restrictions[i][0] - restrictions[i-1][0]
            restrictions[i][1] = min(
                restrictions[i][1],
                restrictions[i-1][1] + d
            )

        for i in range(m-2,-1,-1):
            d = restrictions[i+1][0] - restrictions[i][0]
            restrictions[i][1] = min(
                restrictions[i][1],
                restrictions[i+1][1] + d
            )

        ans = 0

        for i in range(1,m):

            x1,h1 = restrictions[i-1]
            x2,h2 = restrictions[i]

            d = x2 - x1

            ans = max(
                ans,
                (h1+h2+d)//2
            )

        return ans
