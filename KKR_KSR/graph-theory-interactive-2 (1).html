CHAPTER 2

Types of Graphs

Undirected Graph

Edges have no direction. A↔B. Example: Facebook friendships.

Directed Graph (Digraph)

Edges have direction. A→B only. Example: Twitter follow.

Weighted Graph

Edges carry cost or distance. Example: Road distances.

DAG (Directed Acyclic)

Directed graph with no cycles. Example: Task scheduling.

Complete Graph

Every vertex connects to every other vertex.

Bipartite Graph

Vertices split in 2 sets, edges only between sets. Example: Netflix users ↔ movies.

Undirected Graph — Selected
Edges are bidirectional A ↔ B means B ↔ A Example: Social networks

Breadth First Search 🌊

CORE IDEA

Level by Level

BFS explores all neighbors at the current depth before going deeper. Uses a Queue (FIFO).

USES

Applications

Shortest path in unweighted graphs, level-order traversal, web crawling, social network friend suggestions.

Python — BFS 
from collections import deque

graph = {
    "A": ["B", "C"],
    "B": ["D", "E"],
    "C": ["F"],
    "D": [], "E": [], "F": []
}

def bfs(start):
    visited = set()
    queue = deque([start])

    while queue:
        node = queue.popleft()    # FIFO

        if node not in visited:
            print(node, end=" ")
            visited.add(node)

            for neighbor in graph[node]:
                queue.append(neighbor)

bfs("A")
# Output: A B C D E F

Python — Dijkstra 
import heapq

graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'C': 2, 'D': 5},
    'C': {'D': 1},
    'D': {}
}

def dijkstra(start):
    pq = [(0, start)]          # (cost, node)
    dist = {v: float('inf') for v in graph}
    dist[start] = 0

    while pq:
        cost, node = heapq.heappop(pq)

        for neighbor, weight in graph[node].items():
            new_cost = cost + weight

            if new_cost < dist[neighbor]:
                dist[neighbor] = new_cost
                heapq.heappush(pq, (new_cost, neighbor))

    return dist

print(dijkstra('A'))
# {'A': 0, 'B': 1, 'C': 3, 'D': 4}

Python — Topological Sort (DFS) 
from collections import defaultdict

def topological_sort(graph):
    visited = set()
    stack = []

    def dfs(node):
        visited.add(node)
        for neighbor in graph[node]:
            if neighbor not in visited:
                dfs(neighbor)
        stack.append(node)   # Add AFTER all neighbors

    for node in graph:
        if node not in visited:
            dfs(node)

    return stack[::-1]   # Reverse for correct order

Minimum Spanning Tree 🌲

DEFINITION

What is MST?

A subset of edges that connects all V vertices with minimum total weight and no cycles. Always has exactly V−1 edges.

PRIM'S ALGORITHM

Grow from a Node

Start at one vertex. Always pick the cheapest edge that connects a new unvisited vertex. Uses a min-heap.

KRUSKAL'S ALGORITHM

Sort All Edges

Sort every edge by weight. Add edges one by one — skip any that would create a cycle. Uses Union-Find.

REAL WORLD

Why It Matters

Fiber cable layout, power grids, road networks, water pipelines — any problem asking to connect all points cheaply.

DEEP DIVE

How the Min-Heap Works in Prim's

HEAP PROPERTY

Parent ≤ Children

In a min-heap, the smallest element is always at the root (top). Every pop gives you the cheapest available edge in O(log n) time.

PUSH (heappush)

Add to Heap

Insert new edge at end → bubble up until parent ≤ child. O(log n).

POP (heappop)

Extract Minimum

Remove root → move last element to root → sift down to restore heap. O(log n).

CODE

Python Code — Line by Line

Python — Prim's 
import heapq

graph = {
  'A': [('B',4),('C',2),('D',3)],
  'B': [('A',4),('D',5)],
  'C': [('A',2),('D',1)],
  'D': [('A',3),('B',5),('C',1)]
}

def prim(start):
  visited = set()              # Tracks included nodes
  heap = [(0, start, None)]   # (cost, node, from)
  total = 0
  mst = []

  while heap:
    cost, node, frm = heapq.heappop(heap)  # Pop minimum

    if node in visited:   # Already in MST → skip
      continue

    visited.add(node)    # Add to MST
    total += cost

    if frm:
      mst.append((frm, node, cost))

    for neighbor, w in graph[node]:
      if neighbor not in visited:
        heapq.heappush(heap, (w, neighbor, node))

  print("MST:", mst)
  print("Cost:", total)

prim('A')
# MST: [('A','C',2),('C','D',1),('A','B',4)]
# Cost: 7
Python — Kruskal's 
parent = {}

def find(x):          # Path compression
  if parent[x] != x:
    parent[x] = find(parent[x])
  return parent[x]

def union(x, y):      # Merge two sets
  parent[find(y)] = find(x)

edges = [
  ('C','D',1), ('A','C',2), ('A','D',3),
  ('A','B',4), ('B','D',5)
]
vertices = ['A','B','C','D']

for v in vertices:     # Each node is its own parent
  parent[v] = v

edges.sort(key=lambda x: x[2])   # Sort by weight
mst, cost = [], 0

for u, v, w in edges:
  if find(u) != find(v):    # No cycle
    union(u, v)
    mst.append((u, v, w))
    cost += w

print("MST:", mst)
# [('C','D',1),('A','C',2),('A','B',4)]
print("Cost:", cost)   # 7

LINE-BY-LINE EXPLANATION

import heapq

Python's built-in min-heap. Automatically keeps the smallest element at index 0.

heap = [(0, start, None)]

Each entry is (weight, node, from_node). Start with cost 0 for the source.

heappop(heap)

Removes and returns the tuple with minimum weight. This is the greedy choice.

if node in visited: continue

Prevents revisiting nodes already in the MST — this avoids cycles automatically.

heappush(heap, (w, nb, node))

Push all unvisited neighbors with their edge weights. Heap re-sorts automatically.
